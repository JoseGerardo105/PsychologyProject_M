Gracias por preferir a Psynergia - Sistema de Gestion de Citas Para Psicologos V.1.0.

A continuacion, encontrara una guia rapida acerca de la rapida instalacion y ejecucion del sistema:

1. Encontrara 2 carpetas (Front-End y Back-End). Abralas ambas en el IDE de su conveniencia (puede ser VSC).
2. Cuando ya tenga ambas carpetas abiertas en este IDE, abra la consola e instale las dependencias necesarias de ambas carpetas (con el comando "npm i")
Aclaracion: en caso de presentarse un error respecto a las dependencias en el Front-End, ejecute el comando "npm i --force"
3. Una vez hecho esto, estara preparado para ejecutar el sistema. Para ello, en ambos casos se a implementar el comando "npm run dev"
4. Ingrese al siguiente link: http://127.0.0.1:5173/. Asegurese que ambas partes del proyecto este corriendo.

Una vez en el sistema, tenga en cuenta las siguientes consideraciones:

1. A la hora de registrarse y esperar el correo, en caso de no ver el correo en la bandeja principal, revise la carpeta de spam.
2. En la pantalla que muestra la respectiva informacion de los pacientes en las tablas, este mismo se edita de manera inline (haciendo doble click en el campo a modificar y guardandolo con el boton.)
3. De la misma manera, en la misma pantalla de historia, aca solo se edita un dato a la vez. Se va a pulir esta parte en futuras actualizaciones.
4. La busqueda en las pantallas se emplea como criterio de busqueda el documento.
